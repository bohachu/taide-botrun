114學測國文科分析與準備

一、考試科目與成績計算

測驗科目

  • 國語文綜合能力測驗（國綜）：滿分100分
  • 國語文寫作能力測驗（國寫）：滿分50分

成績計算公式

國文總成績 = (國綜得分 × 0.5) + 國寫得分

二、測驗題型與時間

國綜（90分鐘）

  • 選擇題：單選題48分 + 複選題28分 = 76分
  • 混合題：選擇4分 + 簡答與詳答20分 = 24分

國寫（90分鐘）

  • 知性寫作：資訊統整分析 + 觀點表達論說文 = 25分
  • 情意寫作：經驗描寫與反思引導式寫作 = 25分

三、命題趨勢分析

1. 多元文本閱讀

著重不同文類與跨領域文本分析，包含表格、圖形等綜合理解能力

2. A類選文為命題重點

「以15篇核心古文為主，及被各版本選入課本內的文章」，涵蓋字詞義、文法、寫作手法、文化常識等

3. 文言文閱讀

評鑑文法句型、字詞義、國學常識與詩文鑑賞能力

4. 國寫趨勢

  • 知性題：「要求以文字闡述個人觀點，表達個人意見」
  • 情意題：「挑選相符或相似的生命經驗，寫出個人的感受或體會」

四、準備建議

1. 精讀A類選文

包括《禮記》《論語》《孟子》等共15篇核心古文，作為基礎練習

2. 勤寫摘要

同時提升閱讀與寫作能力，訓練精準表達與理解分析能力

3. 分項鍛練寫作

針對不同題型需求，分別訓練擷取分析、批判論述、經驗反思等能力

4. 準備題材素材

提前整理個人生命經驗故事，作為情意題寫作基礎
