### 2. 常見誤讀字系列
- **徜徉**：ㄔㄤˊ ㄧㄤˊ (非 ㄔㄤˇ)
- **踟躕**：ㄔˊ ㄔㄨˊ (非 ㄓˊ)
- **氾濫**：ㄈㄢˋ ㄌㄢˋ
- **湖泊**：ㄏㄨˊ ㄆㄛ (非 ㄆㄛˋ)
- **檔案**：ㄉㄤˋ ㄢˋ (非 ㄉㄤˇ)
- **框架**：ㄎㄨㄤˋ ㄐㄧㄚˋ
- **熾熱**：ㄔˋ ㄖㄜˋ

## 四、發現的問題與修正

### 題目設計問題統計
- **問題題目數**：約8題 (27%)
- **主要問題類型**：
  1. 多個選項都符合正確答案 (如 PH-007)
  2. 所有選項都不符合題幹要求 (如 PH-002)
  3. 干擾選項實際也正確 (如 PH-003, PH-006)
  4. 題型邏輯設計失誤 (如 PH-008, PH-010)

### 需要修正的題目清單

#### PH-002: 所有選項讀音都不同
- **問題**：「何者讀音兩兩相同」,但所有選項都不同
- **建議**：改為「何者讀音不同」或重新設計選項

#### PH-003: 多個選項都正確
- **問題**：A 和 C 的讀音標示都正確
- **建議**：修改錯誤選項,確保只有一個正確答案

#### PH-006: 多個選項都正確
- **問題**：A 和 C 都正確
- **建議**：調整選項設計

#### PH-007: 多個正確答案
- **問題**：B (紓抒舒) 和 C (甄箴斟) 都完全相同
- **建議**：移除其中一個,保留唯一正確答案

#### PH-008: 所有選項都正確
- **問題**：題目要求選出正確讀音,但所有選項都正確
- **建議**：改為「何者錯誤」題型或修改選項

#### PH-009: 多個選項聲母相同
- **問題**：A、B、C 的聲母都相同
- **建議**：修改為只有一個選項符合

#### PH-010: 題型邏輯錯誤
- **問題**：「何者與其他三者不同」,但是 A/D 同組,B/C 同組
- **建議**：改為「何者讀 ㄓㄤˇ」或調整選項

#### PH-018: 標記錯誤但實際正確
- **問題**：標記為「註」讀音錯誤,實際正確
- **建議**：更換為真正錯誤的選項

## 七、學習重點整理

### 學生常見錯誤
1. **聲調混淆**：如湖泊 (ㄆㄛ) 誤讀為 ㄆㄛˋ
2. **聲母遺漏**：如諱 (ㄏㄨㄟˋ) 誤讀為 ㄨㄟˋ
3. **形聲誤導**：如踟 (ㄔˊ) 因「知」聲誤讀為 ㄓˊ
4. **多音字語境判斷錯誤**：如「參差」的「差」讀 ㄘ,非常用的 ㄔㄚ/ㄔㄞ/ㄔㄚˋ

### 教學建議
1. **多音字教學**：依詞義分類,而非僅列舉讀音
2. **形聲字教學**：強調聲符僅為參考,非絕對規則
3. **聲調訓練**：加強四聲辨別,尤其陽平 vs. 上聲
4. **文言字彙累積**：閱讀《古文觀止》時隨時查字典
