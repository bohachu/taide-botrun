# 新課綱114學測【國文科】考試資訊整理

## 一、測驗科目結構
學測國文分為兩科，各占50%：
- **國語文綜合能力測驗（國綜）**
- **國語文寫作能力測驗（國寫）**

## 二、成績計算方式
- 國綜滿分：100分
- 國寫滿分：50分
- **總成績公式**：「(國綜得分 × 0.5) + 國寫得分」

## 三、測驗題型與時間

### 國綜（90分鐘）
| 題型 | 配分 |
|------|------|
| 單選題 | 48分 |
| 複選題 | 28分 |
| 選擇題型 | 4分 |
| 簡答與詳答 | 20分 |

### 國寫（90分鐘）
| 題型 | 配分 |
|------|------|
| 知性寫作題 | 25分 |
| 情意寫作題 | 25分 |

## 四、核心準備重點

**A類選文重要性高**：15篇古文為命題基礎，涵蓋形音義、文法、成詞語應用等。

**國寫命題特色**：知性題強調觀點表達與論說能力；情意題重視個人經驗與感受描述。
