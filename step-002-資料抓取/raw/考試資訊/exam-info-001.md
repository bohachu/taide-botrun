# 學科能力測驗 (Scholastic Aptitude Test) 考試資訊

## 考試目標
根據108課綱核心素養設計，測驗四大方向：
- 基本學科能力
- 銜接大學教育的知能
- 跨情境整合運用與問題解決能力
- "閱讀理解、圖表判讀、證據運用、邏輯推論、分析評價、表達說明"

## 考試科目與時間配置

| 科目 | 時間 |
|------|------|
| 國語文綜合能力測驗(國綜) | 90分鐘 |
| 國語文寫作能力測驗(國寫) | 90分鐘 |
| 英文 | 100分鐘 |
| 數學A | 100分鐘 |
| 數學B | 100分鐘 |
| 社會 | 110分鐘 |
| 自然 | 110分鐘 |

**說明：** 考生可自由選考，國文分兩節施測。

## 測驗範圍 (111學年起依據108課綱)

- **國文**：必修國文
- **英文**：必修英文
- **數學A**：10年級必修+11年級必修數學A類
- **數學B**：10年級必修+11年級必修數學B類
- **社會**：必修歷史、地理、公民與社會
- **自然**：必修物理、化學、生物、地球科學(含探究與實作)

## 題型與計分

**題型包括：**
- 選擇題型(單選、多選)
- 選填題型(數學A、B限定)
- 混合題型(選擇+非選擇)
- 非選擇題(國寫全為非選)

**計分方式：** 各科試卷架構詳見考試說明；國綜與國寫各占國文總成績50%；社會與自然各科試題比例相當。

## 作答方式

111學年起採用「卷卡合一」A3答題卷，整合答案卡與答案卷，考生可同時作答選擇與非選擇題。
