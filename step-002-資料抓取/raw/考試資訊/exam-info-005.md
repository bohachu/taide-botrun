# 112學測考試資訊結構化整理

## 考試說明
新型學測基於108課綱，自111學年度起實施，引入混合題型（選擇題、選填題、非選擇題）與新式A3答題卷。

## 考試科目與測驗範圍

| 科目 | 考試範圍 | 時間 |
|------|---------|------|
| 國文 | 高一、高二、高三（上）必修 | 90分鐘 |
| 英文 | 高一、高二、高三（上）必修 | 100分鐘 |
| 數學A | 高一必修、高二必修數學A類 | 100分鐘 |
| 數學B | 高一必修、高二必修數學B類 | 100分鐘 |
| 自然 | 物理（高一、二）、化學（高一、二）、生物（高一）、地科（高一） | 110分鐘 |
| 社會 | 歷史、地理、公民（均高一、二） | 110分鐘 |

## 5大命題改變

1. **素養與基礎題並重** - 各佔一定比例
2. **情境導向命題** - "主要考核學生能否將知識整合應用在實際情境中"
3. **混合題型設計** - 提升思考層次
4. **非選擇題增加** - 占比逐年提升（國文22%、英文10%、數學15%、社會32%、自然44%）
5. **題量與時間調整** - 國文加10分鐘，其他以歷年標準為主
